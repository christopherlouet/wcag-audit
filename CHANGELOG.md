# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-18

### Added
- SKILL.md with 11 audit categories, axe-core impact levels, workflow, and output format
- references/RULES.md with 64 rules across 11 WCAG categories (images, forms, keyboard, buttons, color, ARIA, structure, tables, frames, deprecated, WCAG 2.2)
- references/PATTERNS.md with regex detection patterns for static code analysis
- references/EXAMPLES.md with good/bad code examples per category (TSX/CSS)
- references/CHECKLIST.md with manual review checklist for needs-review items
- README.md with install instructions, usage examples, and complementary tools
